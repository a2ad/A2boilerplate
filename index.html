A2boilerplate

Biblioteca para auxiliar no pontapé inicial em um projeto web. A idéia foi juntar alguns dos projetos que mais utilizávamos como referência e adaptarmos para um modelo de desenvolvimento da A2 com um Guia de Estilo incluso.

Use o A2boilerplate versão 3.2.4
